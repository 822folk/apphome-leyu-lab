# apphome-leyu-lab

本仓库用于归档和发布多个独立的 HTML 页面。这些页面可作为静态资源直接访问，适用于个人项目、工具页面、实验性网页等场景。

## 目录说明

```
.
├── index.html          # 主入口页面（如有）
├── pages/              # 存放各独立 HTML 页面
├── assets/             # 公共资源（图片、样式、脚本等）
└── README.md           # 本文件
```

- `pages/`：每个子文件夹或文件对应一个独立的 HTML 页面，可包含其所需资源。
- `assets/`：存放多个页面共用的样式文件、脚本或图片，便于统一管理。

## 页面归档说明

- 所有页面均为独立 HTML 文件，可直接在浏览器中打开。
- 页面间无强依赖关系，可单独部署或引用。
- 归档内容不定期更新，新增页面或优化已有页面。

## 使用方式

1. 克隆或下载本仓库。
2. 在浏览器中打开 `index.html` 或 `pages/` 下的任意 HTML 文件。
3. 也可将整个仓库部署至静态托管服务（如 GitHub Pages），获得可直接访问的在线链接。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 来报告问题、建议新页面或改进现有页面。
- 请保持页面结构清晰，资源文件按目录存放。
